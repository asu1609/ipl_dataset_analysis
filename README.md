# ipl_dataset_analysis
IPL Data Analysis using Python &amp; Pandas — exploring match outcomes, toss impact, team performance, and batting stats across seasons.

A exploratory data analysis (EDA) project on IPL cricket data using Python, Pandas, and NumPy. Analyzes two datasets — matches.csv and deliveries.csv — to uncover insights like top-winning teams, toss decision trends, season-wise match counts, and leading run-scorers.
